Edge Cases · Handled vs Deliberately Descoped
✓ Handled
PHONE LinkedIn profile ID (9-digit number) falsely parsed as phone — URLs stripped before phone regex runs.
NAME Filename Preethi_AI_RESUME.pdf extracts "Preethi AI" — domain words blacklisted before filename parsing.
BLEED Cross-candidate data bleed — word-intersection matching prevents "Preethi S" merging with "Naveen V" CSV row.
OCR Invisible zero-width spaces \u200b in PDF text break email regex — stripped at ingestion.
ENCODE Non-UTF8 CSV files — auto-fallback chain: utf-8-sig → latin-1.
⊘ Deliberately Descoped
Scanned image PDFs (OCR) — no text layer extractable; flagged gracefully but not processed.
Live GitHub API rate limiting — stub implementation; real-time scraping out of scope to prioritise robust merging logic.
